# Joke Generator

A fun, lightweight application that delivers a continuous stream of humor through API-driven content.

## Features

- **Random Joke Fetching**: Connects to public joke APIs for fresh content.
- **Glassmorphic UI**: Uses modern CSS effects for a premium look and feel.
- **Instant Refresh**: One-click functionality to generate new jokes.
- **Optimized for Mobile**: Smooth experience across all device types.

## Tech Stack

- **Core**: React, Vite
- **Networking**: Axios / Fetch API
- **Styling**: CSS with Backdrop-filter and Gradients
